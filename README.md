# Aplicação interativa com hand tracking

Hub de projetos com rastreamento de mãos (MediaPipe) no navegador.

### Página inicial (desenvolvedor)
- Coloque sua foto em **`assets/avatar.jpg`** (200×200 px, rosto centralizado).
- Edite nome e subtítulo no cabeçalho de `index.html` (classe `.dev-name` e `.dev-role`).

### Como usar
1. Abra **`index.html`** — página inicial com links para todos os projetos.
2. Ou abra diretamente:
   - **`naruto.html`** — Naruto/Sasuke (mão aberta)
   - **`piano-neon.html`** — Piano neon
   - **`globo-3d.html`** — Globo 3D
   - **`test19.html`** — Partículas 3D / Escreva texto

### Naruto (`naruto.html`)
- **Mão direita aberta:** poder Sasuke
- **Mão esquerda aberta:** poder Naruto

### Partículas 3D (`test19.html`)
- **Padrão:** coração de partículas
- **Digite texto:** até 15 caracteres — as partículas formam o texto
- **Mouse:** afasta as partículas perto do cursor

### Requisitos
- Navegador com câmera e permissão de webcam
- Conexão com internet (CDN MediaPipe / Three.js / texturas)
